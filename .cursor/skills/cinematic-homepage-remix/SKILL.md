# Cinematic Homepage Remix

## Goal
Apply one visual preset to `index.html` without changing page structure.

## Presets
- dawn: soft blue + pink, light glow
- dusk: purple + amber, warm glass
- starlight: deep navy + neon lavender glow
- cyber-mist: cyan + magenta, stronger bloom

## Rules
1. Only edit style-related parts (CSS variables, gradients, shadows, transitions).
2. Keep HTML structure and content unchanged unless user requests.
3. Preserve readability and mobile layout.
4. After edits, briefly list what visual knobs were changed.

## Output style
- Report: selected preset + 4-6 changed style points.
